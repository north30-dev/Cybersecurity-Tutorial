# 网络安全学习教程

> 基于2026年最新技术趋势，融合AI安全与Java代码审计的全流程学习路线

---

## 教程简介

本教程是一套面向大学生的网络安全学习体系，以**Java代码审计**为核心特色，融合**AI安全**前沿技术，构建从入门到就业的完整学习路径。

### 核心特色

- **Java代码审计导向**：以Java生态安全为核心，覆盖Spring全家桶、中间件、反序列化等核心主题
- **AI安全前沿**：涵盖Prompt注入、智能体安全、LLM应用安全测试等2026年最新威胁
- **AI辅助学习**：每个阶段融入AI工具使用，提升学习效率
- **实战驱动**：靶场练习、SRC实战、漏洞复现三位一体
- **就业导向**：从技能培养到简历面试，全流程就业支持

### 适用人群

| 人群特征 | 学习建议 |
|---------|---------|
| 有Java基础的大学生 | Java部分可快速浏览，重点学习安全相关内容 |
| 会Linux基础命令 | Linux操作部分简短，聚焦安全场景应用 |
| 会装虚拟机 | 虚拟机安装简述，重点在安全环境配置 |
| 零基础但有强烈兴趣 | 从筑基篇开始，按路线图循序渐进 |
| 计划从事安全工作 | 完整学习五个阶段，积累实战经验 |

### 阶段详解

#### 阶段一：筑基篇（高考暑假，2-3个月）

**阶段目标**：建立安全思维框架，掌握编程与网络基础知识

| 模块 | 学习内容 | 预计时长 |
|-----|---------|---------|
| Java安全基础 | 安全编码规范、常见漏洞模式、反序列化基础 | 2周 |
| Linux安全操作 | 权限管理、日志分析、系统加固 | 1周 |
| 网络协议安全 | TCP/IP、HTTP/HTTPS、DNS安全 | 3周 |
| Web安全入门 | OWASP项目、核心漏洞类型、测试思维 | 2周 |

#### 阶段二：打靶篇（大一上学期，4个月）

**阶段目标**：通过靶场实战培养渗透测试思维，入门代码审计

| 模块 | 学习内容 | 预计时长 |
|-----|---------|---------|
| 靶场环境搭建 | DVWA、Pikachu、PortSwigger Academy | 2周 |
| 信息收集技术 | 域名枚举、端口扫描、指纹识别、搜索引擎利用 | 4周 |
| Java代码审计入门 | Semgrep、CodeQL、AI辅助审计 | 5周 |
| 安全脚本编写 | Python安全脚本、POC编写、自动化工具 | 3周 |

#### 阶段三：进阶篇（大一下学期，4个月）

**阶段目标**：深入理解漏洞原理，开始SRC实战挖掘

| 模块 | 学习内容 | 预计时长 |
|-----|---------|---------|
| 渗透测试全流程 | 完整方法论、报告编写、AI辅助分析 | 4周 |
| SRC实战指南 | 平台选择、挖掘技巧、报告规范 | 5周 |
| 中间件安全 | WebLogic、Shiro、Fastjson、Log4j2 | 4周 |
| OWASP Top10 2026 | 最新Top10详解、供应链安全 | 3周 |

#### 阶段四：核心篇（大二上学期，4个月）

**阶段目标**：掌握内网渗透、云原生安全、AI安全三大核心方向

| 模块 | 学习内容 | 预计时长 |
|-----|---------|---------|
| 内网渗透 | 域环境、横向移动、域控攻击 | 5周 |
| 云原生安全 | Docker安全、K8s安全、CI/CD安全 | 4周 |
| AI安全 | Prompt注入、智能体安全、LLM测试 | 5周 |

#### 阶段五：就业篇（大二下学期，4个月）

**阶段目标**：完成就业准备，积累实战经验

| 模块 | 学习内容 | 预计时长 |
|-----|---------|---------|
| 安全开发实践 | SDL流程、威胁建模、安全框架 | 3周 |
| 安全合规基础 | 等保、ISO27001、数据安全法 | 2周 |
| 简历与面试准备 | 岗位分析、简历编写、面试题库 | 3周 |
| 实习实战经验 | 实习准备、工作适应、职业规划 | 4周 |

## 学习建议

### 学习原则

1. **循序渐进**：严格按照阶段顺序学习，不要跳跃
2. **动手实践**：每个知识点都要在靶场或环境中验证
3. **总结记录**：建立学习笔记，记录踩坑经验
4. **合法合规**：仅在授权范围内进行安全测试

### 每日学习计划建议

| 时间段 | 学习内容 | 时长 |
|-------|---------|------|
| 理论学习 | 阅读教程文档，理解原理 | 1-2小时 |
| 实践操作 | 靶场练习、工具使用 | 2-3小时 |
| 总结复盘 | 整理笔记、撰写博客 | 0.5-1小时 |

### 学习资源使用

- **本文档**：作为学习主线，按章节顺序学习
- **附录A**：工具使用时随时查阅命令手册
- **附录B**：漏洞复现时快速搭建环境
- **附录C**：深入学习时参考推荐资源
- **附录D**：学习AI辅助技巧提升效率

### 常见问题解答

**Q: 没有编程基础可以学吗？**
A: 可以，但建议先学习Python基础（2周），再开始筑基篇学习。

**Q: 学习周期可以缩短吗？**
A: 建议按推荐周期学习。如果基础较好，可以适当压缩筑基篇时间。

**Q: 如何平衡学习与学校课程？**
A: 本教程设计已考虑大学时间安排，每阶段对应一个学期，合理规划即可。

**Q: 学完能找到工作吗？**
A: 完整学完五个阶段，配合SRC实战经验和CTF成绩，具备初级安全岗位能力。
