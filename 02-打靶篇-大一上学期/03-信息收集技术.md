# 信息收集技术

## 一、信息收集概述

### 1.1 什么是信息收集

信息收集（Information Gathering），又称侦察（Reconnaissance），是渗透测试的第一阶段，也是最关键的阶段。通过收集目标系统的各种信息，为后续的漏洞发现和利用奠定基础。

**核心原则**：
- 信息收集占渗透测试60%-70%的时间
- 信息越多，攻击面越大，成功概率越高
- 被动收集优先，减少被发现的概率

### 1.2 信息收集分类

#### 按交互方式分类

| 类型 | 特点 | 风险 | 示例 |
|------|------|------|------|
| 被动收集 | 不与目标直接交互 | 低 | 搜索引擎、WHOIS、证书透明度 |
| 主动收集 | 与目标直接交互 | 高 | 端口扫描、目录枚举、指纹识别 |

#### 按收集内容分类

```
信息收集内容：
├── 域名信息
│   ├── 子域名
│   ├── DNS记录
│   ├── 域名历史
│   └── 证书信息
├── IP与端口
│   ├── IP段发现
│   ├── 存活主机
│   ├── 开放端口
│   └── 服务识别
├── Web应用
│   ├── 技术栈识别
│   ├── 目录结构
│   ├── 敏感文件
│   └── 指纹信息
├── 人员信息
│   ├── 邮箱地址
│   ├── 社交账号
│   ├── 组织架构
│   └── 泄露信息
└── 其他信息
    ├── 历史漏洞
    ├── 关联资产
    └── 供应链信息
```

### 1.3 法律边界

**重要提醒**：
- 仅对授权目标进行信息收集
- 被动信息收集相对安全，但仍需谨慎
- 主动扫描可能触发告警，需在授权范围内进行
- 遵守当地法律法规和道德准则

### 2.2 DNS记录分析

#### DNS记录类型

| 记录类型 | 作用 | 安全意义 |
|----------|------|----------|
| A | 域名→IPv4地址 | 发现真实IP |
| AAAA | 域名→IPv6地址 | 发现IPv6资产 |
| CNAME | 域名别名 | 发现关联域名 |
| MX | 邮件服务器 | 发现邮件系统 |
| NS | 域名服务器 | 发现DNS架构 |
| TXT | 文本记录 | 发现SPF、验证信息 |
| SRV | 服务记录 | 发现内部服务 |
| SOA | 起始授权 | 管理员邮箱 |

#### dig命令详解

```bash
# 基本查询
dig example.com

# 指定记录类型
dig example.com A
dig example.com MX
dig example.com TXT
dig example.com ANY

# 指定DNS服务器
dig @8.8.8.8 example.com

# 反向查询
dig -x 192.168.1.1

# DNS区域传输（如果允许）
dig @ns1.example.com example.com AXFR

# 追踪DNS解析过程
dig +trace example.com

# 短格式输出
dig +short example.com

# 批量查询
for domain in $(cat domains.txt); do
    dig +short $domain
done
```

### 4.5 自定义指纹识别

```python
#!/usr/bin/env python3
import requests
import re

def identify_fingerprint(url):
    """Web指纹识别"""
    fingerprints = {
        'WordPress': [
            r'wp-content',
            r'wp-includes',
            r'WordPress',
        ],
        'Drupal': [
            r'Drupal',
            r'drupal.js',
        ],
        'Joomla': [
            r'Joomla',
            r'/components/',
        ],
        'ThinkPHP': [
            r'ThinkPHP',
            r'think_show_page_trace',
        ],
        'Laravel': [
            r'Laravel',
            r'laravel_session',
        ],
        'Spring': [
            r'Spring',
            r'Whitelabel Error Page',
        ],
        'Django': [
            r'csrfmiddlewaretoken',
            r'__admin_media_prefix__',
        ],
        'Flask': [
            r'Flask',
            r'werkzeug',
        ],
    }
    
    try:
        resp = requests.get(url, timeout=10)
        content = resp.text
        headers = str(resp.headers)
        
        results = []
        for cms, patterns in fingerprints.items():
            for pattern in patterns:
                if re.search(pattern, content, re.I) or re.search(pattern, headers, re.I):
                    results.append(cms)
                    break
        
        return list(set(results))
    except Exception as e:
        return [f"Error: {e}"]

# 使用示例
if __name__ == "__main__":
    url = "http://example.com"
    result = identify_fingerprint(url)
    print(f"识别结果: {result}")
```

#### 敏感文件扫描脚本

```python
#!/usr/bin/env python3
import requests
import concurrent.futures

sensitive_paths = [
    # Git相关
    '/.git/config',
    '/.git/HEAD',
    '/.gitignore',
    # SVN相关
    '/.svn/entries',
    # 配置文件
    '/.env',
    '/config.php',
    '/configuration.php',
    '/settings.py',
    '/web.config',
    '/phpinfo.php',
    '/info.php',
    # 备份文件
    '/backup.zip',
    '/backup.sql',
    '/db.sql',
    '/database.sql',
    '/www.zip',
    '/web.tar.gz',
    '/1.zip',
    '/backup/',
    # 管理后台
    '/admin/',
    '/admin.php',
    '/administrator/',
    '/manage/',
    '/manager/',
    '/phpmyadmin/',
    # 其他
    '/robots.txt',
    '/sitemap.xml',
    '/.DS_Store',
    '/server-status',
    '/server-info',
]

def check_path(url, path):
    try:
        resp = requests.get(url + path, timeout=5, allow_redirects=False)
        if resp.status_code in [200, 301, 302, 403]:
            return path, resp.status_code
    except:
        pass
    return None

def scan_sensitive_files(url):
    results = []
    with concurrent.futures.ThreadPoolExecutor(max_workers=20) as executor:
        futures = {executor.submit(check_path, url, path): path for path in sensitive_paths}
        for future in concurrent.futures.as_completed(futures):
            result = future.result()
            if result:
                results.append(result)
    return results

if __name__ == "__main__":
    url = "http://example.com"
    found = scan_sensitive_files(url)
    for path, status in found:
```

### 8.2 自定义自动化脚本

```python
#!/usr/bin/env python3
"""
自动化信息收集脚本
整合多个工具进行一站式收集
"""
import subprocess
import json
import requests
from concurrent.futures import ThreadPoolExecutor
import argparse

class InfoGatherer:
    def __init__(self, domain):
        self.domain = domain
        self.results = {
            'subdomains': [],
            'ips': [],
            'ports': {},
            'fingerprints': {},
            'emails': [],
            'sensitive_paths': []
        }
    
    def subdomain_enum(self):
        """子域名枚举"""
        print("[*] 开始子域名枚举...")
        try:
            # 使用subfinder
            result = subprocess.run(
                ['subfinder', '-d', self.domain, '-silent'],
                capture_output=True, text=True
            )
            subdomains = result.stdout.strip().split('\n')
            self.results['subdomains'] = [s for s in subdomains if s]
            print(f"[+] 发现 {len(self.results['subdomains'])} 个子域名")
        except Exception as e:
            print(f"[-] 子域名枚举失败: {e}")
    
    def dns_resolve(self):
        """DNS解析"""
        print("[*] 开始DNS解析...")
        for subdomain in self.results['subdomains']:
            try:
                result = subprocess.run(
                    ['dig', '+short', subdomain],
                    capture_output=True, text=True
                )
                ips = result.stdout.strip().split('\n')
                for ip in ips:
                    if ip and ip not in self.results['ips']:
                        self.results['ips'].append(ip)
            except:
                pass
        print(f"[+] 发现 {len(self.results['ips'])} 个IP地址")
    
    def port_scan(self):
        """端口扫描"""
        print("[*] 开始端口扫描...")
        if not self.results['ips']:
            return
        
        targets = self.results['ips'][:10]  # 限制扫描数量
        for ip in targets:
            try:
                result = subprocess.run(
                    ['nmap', '-sS', '-sV', '-T4', '--top-ports', '100', ip],
                    capture_output=True, text=True
                )
                self.results['ports'][ip] = result.stdout
            except:
                pass
    
    def fingerprint_scan(self):
        """指纹识别"""
        print("[*] 开始指纹识别...")
        for subdomain in self.results['subdomains'][:20]:
            try:
                result = subprocess.run(
                    ['whatweb', '-v', f'http://{subdomain}'],
                    capture_output=True, text=True, timeout=30
                )
                self.results['fingerprints'][subdomain] = result.stdout
            except:
                pass
    
    def email_collect(self):
        """邮箱收集"""
        print("[*] 开始邮箱收集...")
        try:
            result = subprocess.run(
                ['theharvester', '-d', self.domain, '-b', 'google', '-l', '100'],
                capture_output=True, text=True
            )
            # 解析结果提取邮箱
            lines = result.stdout.split('\n')
            for line in lines:
                if '@' in line and self.domain in line:
                    self.results['emails'].append(line.strip())
        except Exception as e:
            print(f"[-] 邮箱收集失败: {e}")
    
    def run(self):
        """执行所有收集任务"""
        print(f"\n{'='*50}")
        print(f"目标: {self.domain}")
        print(f"{'='*50}\n")
        
        # 并行执行
        with ThreadPoolExecutor(max_workers=3) as executor:
            executor.submit(self.subdomain_enum)
            executor.submit(self.email_collect)
        
        # 顺序执行（依赖前面的结果）
        self.dns_resolve()
        self.port_scan()
        self.fingerprint_scan()
        
        # 输出结果
        self.save_results()
    
    def save_results(self):
        """保存结果"""
        filename = f"{self.domain}_recon.json"
        with open(filename, 'w') as f:
            json.dump(self.results, f, indent=2, ensure_ascii=False)
        print(f"\n[+] 结果已保存到 {filename}")

def main():
    parser = argparse.ArgumentParser(description='自动化信息收集')
    parser.add_argument('-d', '--domain', required=True, help='目标域名')
    args = parser.parse_args()
    
    gatherer = InfoGatherer(args.domain)
    gatherer.run()

if __name__ == '__main__':
    main()
```

```python
#!/usr/bin/env python3
"""
信息收集报告生成器
"""
import json
from datetime import datetime

class ReconReport:
    def __init__(self, domain):
        self.domain = domain
        self.timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        self.data = {}
    
    def load_results(self, files):
        """加载各类结果文件"""
        for name, filepath in files.items():
            try:
                with open(filepath, 'r') as f:
                    if filepath.endswith('.json'):
                        self.data[name] = json.load(f)
                    else:
                        self.data[name] = f.read().splitlines()
            except:
                self.data[name] = []
    
    def generate_report(self):
        """生成Markdown报告"""
        report = f"""# 信息收集报告

## 基本信息
- 目标域名: {self.domain}
- 收集时间: {self.timestamp}

## 子域名 ({len(self.data.get('subdomains', []))}个)
"""
        for sub in self.data.get('subdomains', [])[:50]:
            report += f"- {sub}\n"
        
        report += f"""
## 开放端口
```
{self.data.get('ports', 'N/A')}
```

## 技术栈
```
{self.data.get('fingerprints', 'N/A')}
```

## 邮箱地址 ({len(self.data.get('emails', []))}个)
"""
        for email in self.data.get('emails', []):
            report += f"- {email}\n"
        
        report += """
## 风险评估

### 高风险发现
- [ ] 敏感文件泄露
- [ ] 弱口令服务
- [ ] 过时组件版本

### 中风险发现
- [ ] 信息泄露
- [ ] 配置不当

### 建议措施
1. 修复敏感信息泄露
2. 更新组件版本
3. 加强访问控制
"""
        
        return report
    
    def save_report(self, filename):
        """保存报告"""
        report = self.generate_report()
        with open(filename, 'w') as f:
            f.write(report)
        print(f"[+] 报告已保存: {filename}")

# 使用示例
if __name__ == "__main__":
    report = ReconReport("example.com")

        'subdomains': 'subdomains.txt',

    })
```
