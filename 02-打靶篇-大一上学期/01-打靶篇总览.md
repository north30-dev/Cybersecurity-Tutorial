# 打靶篇总览

> 纸上得来终觉浅，绝知此事要躬行。打靶篇通过靶场实战培养渗透测试思维。

---

## 阶段目标

打靶篇是理论到实践的桥梁，目标是：

1. **搭建靶场环境**：DVWA、Pikachu、PortSwigger Academy等
2. **掌握信息收集**：域名枚举、端口扫描、指纹识别
3. **入门代码审计**：Semgrep、CodeQL、AI辅助审计
4. **编写安全脚本**：Python脚本、POC开发、自动化工具

**学习时长**：大一上学期，约4个月

## 靶场通关目标

### DVWA通关清单

| 模块 | Low | Medium | High | Impossible |
|-----|-----|--------|------|-----------|
| Brute Force | 爆破攻击 | 爆破绕过 | 高效爆破 | 理解防御 |
| Command Injection | 命令注入 | 过滤绕过 | 复杂绕过 | 安全实现 |
| SQL Injection | 注入利用 | 盲注技巧 | 高级绕过 | 预编译防御 |
| XSS (Reflected) | 反射XSS | 过滤绕过 | DOM XSS | 编码防御 |
| XSS (Stored) | 存储XSS | 存储绕过 | 高级利用 | 输出编码 |
| File Upload | 文件上传 | 类型绕过 | 重写绕过 | 严格验证 |

### PortSwigger Academy学习路径

推荐按以下顺序学习实验室：

1. **SQL注入路径**：基础注入 → 盲注 → 高级注入
2. **XSS路径**：反射型 → 存储型 → DOM型
3. **CSRF路径**：基础CSRF → 绕过技巧
4. **认证路径**：暴力破解 → 会话管理
5. **访问控制路径**：IDOR → 权限绕过

## 常见问题

**Q: 靶场通关后没有成就感？**
A: 尝试不看Writeup独立通关，或挑战更高难度的靶场如Hack The Box。

**Q: 信息收集工具太多，学哪个？**
A: 先精通Nmap和Burp Suite，其他工具按需学习。

**Q: 代码审计感觉很难？**
A: 先从简单漏洞模式开始，配合AI辅助理解，逐步深入。

**Q: 学完打靶篇能达到什么水平？**
A: 能够独立完成基础渗透测试，入门代码审计，具备初级安全能力。
