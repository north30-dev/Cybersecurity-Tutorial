# 虚拟机环境搭建

本文档简述网络安全学习所需的虚拟机环境搭建，包括VMware安装、Kali Linux配置、Windows靶机配置及网络模式说明。

---

## 一、VMware安装

### 1.1 下载地址

- **VMware Workstation Pro**（Windows）：https://www.vmware.com/products/workstation-pro.html
- **VMware Fusion**（macOS）：https://www.vmware.com/products/fusion.html

> 提示：VMware Workstation Pro 现已免费供个人使用，可直接下载安装。

### 1.2 安装步骤

1. 下载对应系统的安装包
2. 运行安装程序，选择典型安装
3. 安装完成后重启系统
4. 输入许可证密钥（个人使用可跳过）

## 二、Kali Linux配置

### 2.1 镜像下载

- 官方下载：https://www.kali.org/get-kali/#kali-installer-images
- 推荐选择：**Kali Linux 64-bit (Installer)** ISO镜像
- 国内镜像：https://mirrors.tuna.tsinghua.edu.cn/kali-images/

### 2.2 虚拟机创建

1. 打开VMware，选择"创建新的虚拟机"
2. 选择"典型(推荐)"配置
3. 选择已下载的Kali ISO镜像
4. 设置虚拟机名称和存储位置
5. 配置硬件规格：
   - 内存：建议4GB以上
   - 处理器：2核以上
   - 硬盘：50GB以上
   - 网络：NAT模式

### 2.3 系统安装

1. 启动虚拟机，选择"Graphical Install"
2. 设置语言、时区、键盘布局
3. 设置用户名和密码
4. 分区选择"使用整个磁盘"
5. 安装GRUB引导程序
6. 重启完成安装

### 2.4 基础配置

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 配置国内软件源（可选）
sudo nano /etc/apt/sources.list
# 添加清华源
deb https://mirrors.tuna.tsinghua.edu.cn/kali kali-rolling main contrib non-free

# 安装VMware Tools（增强功能）
sudo apt install open-vm-tools open-vm-tools-desktop -y

# 安装常用工具
sudo apt install git curl wget vim net-tools -y

# 配置Python环境
sudo apt install python3 python3-pip -y
```

### 3.2 Windows靶机基础配置

```powershell
# 关闭防火墙（仅限靶机环境）
netsh advfirewall set allprofiles state off

# 开启远程桌面
reg add "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Terminal Server" /v fDenyTSConnections /t REG_DWORD /d 0 /f

# 创建测试用户
net user test test123 /add
net localgroup administrators test /add
```

## 四、网络模式说明

### 4.1 NAT模式（推荐）

- **特点**：虚拟机通过宿主机NAT转换访问外网
- **优点**：配置简单，宿主机与虚拟机可互相访问
- **适用场景**：日常学习、工具下载
- **IP段**：默认192.168.x.0/24（由VMware分配）

### 4.2 桥接模式

- **特点**：虚拟机直接连接物理网络
- **优点**：获得独立IP，可被同网段其他主机访问
- **适用场景**：内网渗透测试、多机协作
- **注意**：需确保物理网卡正常工作

### 4.3 仅主机模式（Host-Only）

- **特点**：虚拟机仅与宿主机通信，无法访问外网
- **优点**：隔离安全，适合搭建封闭靶场
- **适用场景**：敏感实验、隔离环境测试

### 4.4 网络配置命令

```bash
# Kali Linux网络配置
sudo nano /etc/network/interfaces

# 静态IP配置示例
auto eth0
iface eth0 inet static
    address 192.168.1.100
    netmask 255.255.255.0
    gateway 192.168.1.1

# 重启网络服务
sudo systemctl restart networking

# 查看网络配置
ip addr show
```

### 4.5 网络模式切换

在VMware中：
1. 右键虚拟机 → 设置 → 网络适配器
2. 选择所需网络模式
3. 确认后重启虚拟机网络服务

## 六、常见问题

### Q1: 虚拟机无法联网
检查网络模式是否正确，NAT模式下确保VMware NAT服务已启动。

### Q2: VMware Tools安装失败
使用命令安装开源版本：`sudo apt install open-vm-tools`

### Q3: 虚拟机性能卡顿
- 增加内存分配
- 启用硬件加速
- 关闭不必要的后台服务

---

> **安全提示**：靶机环境务必与生产网络隔离，避免误操作影响真实环境。
