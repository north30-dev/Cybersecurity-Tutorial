#### 2.1.2 自建代码审计流程

**使用LLM API进行代码审计**

```python
import openai
import anthropic

def audit_code(code_snippet, vuln_type):
    """
    使用LLM审计代码
    """
    prompt = f"""
    作为安全审计专家，请分析以下代码是否存在{vuln_type}漏洞。
    
    代码：
    ```
    {code_snippet}
    ```
    
    请提供：
    1. 是否存在漏洞（是/否）
    2. 漏洞位置
    3. 利用方式
    4. 修复建议
    """
    
    client = anthropic.Anthropic()
    response = client.messages.create(
        model="claude-3-opus-20240229",
        max_tokens=4096,
        messages=[{"role": "user", "content": prompt}]
    )
    
    return response.content[0].text

# 使用示例
code = """
def get_user(user_id):
    query = f"SELECT * FROM users WHERE id = {user_id}"
    return db.execute(query)
"""

result = audit_code(code, "SQL注入")
print(result)
```

**批量代码审计脚本**

```python
import os
import anthropic
from pathlib import Path

class CodeAuditor:
    def __init__(self, api_key):
        self.client = anthropic.Anthropic(api_key=api_key)
        self.vuln_types = [
            "SQL注入", "XSS", "命令注入", 
            "路径遍历", "SSRF", "反序列化"
        ]
    
    def audit_file(self, file_path):
        """审计单个文件"""
        with open(file_path, 'r') as f:
            code = f.read()
        
        prompt = f"""
        请对以下代码进行全面安全审计：
        
        文件：{file_path}
        代码：
        ```
        {code}
        ```
        
        请检查以下漏洞类型：
        {', '.join(self.vuln_types)}
        
        输出格式：
        - 漏洞类型 | 位置 | 严重程度 | 修复建议
        """
        
        response = self.client.messages.create(
            model="claude-3-opus-20240229",
            max_tokens=4096,
            messages=[{"role": "user", "content": prompt}]
        )
        
        return response.content[0].text
    
    def audit_project(self, project_path, extensions=['.py', '.java', '.php', '.js']):
        """审计整个项目"""
        results = {}
        for ext in extensions:
            for file_path in Path(project_path).rglob(f'*{ext}'):
                # 跳过测试文件和依赖
                if 'test' in str(file_path).lower() or 'node_modules' in str(file_path):
                    continue
                try:
                    results[str(file_path)] = self.audit_file(file_path)
                except Exception as e:
                    results[str(file_path)] = f"Error: {str(e)}"
        return results

# 使用示例
auditor = CodeAuditor(api_key="your-api-key")
results = auditor.audit_project("/path/to/project")
```

## 四、AI辅助安全工作最佳实践

### 4.1 使用原则

#### 4.1.1 安全原则

**数据保护**
- 不要将敏感数据（API密钥、密码、真实IP等）发送给AI
- 不要上传完整的配置文件或数据库文件
- 对代码中的敏感信息进行脱敏处理

```python
# 错误示例 - 包含敏感信息
password = "MySecretPassword123"
api_key = "sk-xxxxxxxxxxxx"

# 正确示例 - 脱敏处理
password = "REDACTED"
api_key = "<API_KEY>"
```

**授权合规**
- 仅在授权范围内使用AI辅助工具
- 遵守组织的AI使用政策
- 记录AI辅助生成的内容

#### 4.1.2 验证原则

**不盲目信任AI输出**
- 所有AI生成的代码需要人工审核
- AI建议的安全修复需要验证有效性
- AI生成的PoC需要在隔离环境测试

**验证流程**

```
AI输出 → 人工审核 → 测试验证 → 应用实施
         ↓           ↓
      发现问题    确认有效
         ↓           ↓
      反馈优化    文档记录
```

#### 4.4.3 成本控制

**API使用优化**
- 缓存常用Prompt结果
- 批量处理减少调用次数
- 选择合适的模型（平衡成本和效果）
- 监控API使用量

**示例**

```python
import hashlib
import json

def cached_ai_call(prompt, cache_file="ai_cache.json"):
    """带缓存的AI调用"""
    # 计算prompt的hash作为缓存key
    key = hashlib.md5(prompt.encode()).hexdigest()
    
    # 尝试从缓存读取
    try:
        with open(cache_file, 'r') as f:
            cache = json.load(f)
            if key in cache:
                return cache[key]
    except FileNotFoundError:
        cache = {}
    
    # 调用AI
    result = actual_ai_call(prompt)
    
    # 保存到缓存
    cache[key] = result
    with open(cache_file, 'w') as f:
        json.dump(cache, f)
    
    return result
```
